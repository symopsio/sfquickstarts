# assets/

Place quickstart image assets here. File names must be lowercase with hyphens (no underscores or special characters).

Referenced in the guide:

- `bronze-glue-databases.png` — Glue Data Catalog databases list
- `bronze-glue-database-detail.png` — Glue database details (LocationUri)
- `bronze-glue-tables-list.png` — Glue tables list showing balloon_game_events
- `bronze-glue-table-iceberg-detail.png` — Glue Iceberg table detail
- `aws-lf-database-settings.png` — Lake Formation edit database (LF mode for new tables)
- `aws-lf-dataa-lake-settings.png` — Lake Formation data lake storage settings
- `aws-lf-data-permissions.png` — Lake Formation data permissions for Snowflake catalog role

Source images are in `lab/images/`. Copy and rename them to match the filenames above before publishing to sfquickstarts.
